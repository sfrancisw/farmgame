# farmgame
[WORK IN PROGRESS]
Walk around the world and fight the animals. Try not to die! Also, pay a visit to the undeground.


CONTROLS:

A/D: Left/Right

W/S: Climb Ladder Up/Down

E: Shrink/Cancel Shrinking

SPACE: Shoot Arrow

ESC: Exit Game
